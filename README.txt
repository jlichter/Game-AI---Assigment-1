Single Agent Movement 
Patrick Butler, Jessica Lichter

The keys are 0 - 6, and correspond to the following algorithms:
0 - showcases Evade() with a pursuing agent using Pursue().
1 - showcases Arrive(). Agent approaches and slows down.
2 - showcases Seek() and Face(). The agent seeks and faces target.
3 - showcases Flee() with a seeking agent using Seek().
4 - showcases Align(). Target spins and agent spins to match orientation of target.
5 - showcases Face(). Target wanders and the agent turns to face target.
6 - showcases Wander(). Agent wanders around.
